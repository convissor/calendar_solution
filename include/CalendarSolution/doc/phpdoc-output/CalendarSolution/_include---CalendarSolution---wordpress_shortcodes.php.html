Calendar Solution's API for WordPress Shortcodes

WordPress has Shortcode functionality built in as of version 2.5.

  • calendar_solution_calendar: calendar_solution_calendar()
  • calendar_solution_detailtable: calendar_solution_detailtable()
  • calendar_solution_list: calendar_solution_list()
  • calendar_solution_monthtitle: calendar_solution_monthtitle()
  • calendar_solution_quicktable: calendar_solution_quicktable()
  • calendar_solution_title: calendar_solution_title()
  • calendar_solution_ul: calendar_solution_ul()

The Attributes available for use with these Shortcodes are...

  • Attributes to get data by ID:
    • category_id: CalendarSolution_List::set_category_id()
    • frequent_event_id: CalendarSolution_List::set_frequent_event_id()
    • page_id: CalendarSolution_List::set_page_id()
  • Attributes to limit the number of results:
    • limit: CalendarSolution_List::set_limit()
  • Attributes to get data a user is seeking:
    • request_properties: CalendarSolution_List::set_request_properties()
  • Attributes for displaying navigation elements:
    • date_navigation_top / date_navigation_bottom: CalendarSolution_List::get_date_navigation()
    • limit_form_top / limit_form_bottom: CalendarSolution_List::get_limit_form()
    • limit_navigation_bottom: CalendarSolution_List::get_limit_navigation()
  • Attributes for formatting data:
    • date_format: CalendarSolution_List::set_date_format()
    • time_format: CalendarSolution_List::set_time_format()
    • show_cancelled: CalendarSolution_List::set_show_cancelled()
    • is_own_event: CalendarSolution_List::set_is_own_event()
    • show_own_events_first: CalendarSolution_List::set_show_own_events_first()
  • Attributes for limiting data by date (the defaults are sane; use only when necessary):
    • from: CalendarSolution_List::set_from()
    • to: CalendarSolution_List::set_to()
    • permit_history_months: CalendarSolution_List::set_permit_history_months()
    • permit_future_months: CalendarSolution_List::set_permit_future_months()

 # Examples
 # MonthTitle, date dropdown list on top, date navigation on bottom.
 [calendar_solution_monthtitle limit_form_top="datelist" date_navigation_bottom=""]

 # QuickTable, five rows, navigation for more events on the bottom.
 [calendar_solution_quicktable frequent_event_id="6" limit="5,null" limit_navigation_bottom=""]
